Code 2
Parsons Design and Technology Final Project

Throughout the semester students focused on animation, interactive typography, camera input, machine learning, shaders, music visualization, and external APIs all using p5.js. Due to Covid the class had one week to concept, design, and develop an entire website around the following prompt. I was blown away with what they created. Hope you enjoy!

Create a single purpose tool

Build and Destroy
By Jiyoon Moon
A web app where the user can write about a minor inconvenience that has plagued their lives, and destroy the text through clapping. The program asks the user to input a problem and then uses the p5 sound library to reduce the “health” of the text until it has gone away. I find it a satisfying feeling to let out annoyance and anger through the destruction of an object, and this program is intended to be a fun and harmless illustration of that.
Weathering with Mars
By Elena Peng
A voice-activated Mars weather assistance, using the P5speech.js library and the information from NASA Insight API. With a simplistic yet animated illustrations, the bot will travel with you to Mars and chat with you about the temperature, time, wind-speed and more information about Mars. This project mainly targets children and younger teenagers, being envisioned in settings like museum interactive displays or educational websites.
Paint With Sound
By Ken Chen
A program that visualizes sound into rings and circles and paints them on a revolving canvas to generate a unique artwork to that sound. Audience can find a different way to visualize the progression of a piece of music or sound than on linear sheet music.
Disko App
By Isaac Sante
Use real-time graphics at your next DJ session with the Disko-App. Just activate the mic and start moving to change colors. Colors react to your body position and sound also generates a graphic. Used machine learning to train model for colors.
My Keyboard
By Jing Dong
A simple web app made for everyone who cannot get access to musical instruments during this difficult time. We should keep creating interesting things even we are trapped in our room.
